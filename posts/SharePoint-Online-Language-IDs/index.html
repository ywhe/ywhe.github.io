SharePoint Online Language IDs

Apr 1, 2023 2 min readSharePoint

SharePoint Online Language ID is a unique identifier that represents a language used in SharePoint Online. Each language has its own ID that is used to indicate the language of the content stored in SharePoint Online. Language IDs are used to ensure that content is displayed correctly in the correct language in SharePoint Online sites and pages.

The Language ID is a numerical value assigned to a specific language. For example, the Language ID for English (United States) is 1033, while the Language ID for Spanish (Spain) is 3082. SharePoint Online supports many different language IDs, and users can configure the language settings for their SharePoint Online sites to ensure that content is displayed in the correct language for their needs.
