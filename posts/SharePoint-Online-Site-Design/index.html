What is SharePoint Online Site Design

Site design refers to the process of customizing and branding the appearance and functionality of SharePoint Online sites to meet specific business needs. This involves creating a design template that can be applied to new or existing sites, which can include custom themes, layouts, and functionality.

Site design is typically performed by SharePoint administrators or developers using tools such as SharePoint Designer or Microsoft Power Platform. It can involve creating custom web parts, adding custom scripts, and implementing custom workflows to automate business processes.

The ultimate goal of Site design is to create a visually appealing and user-friendly site that enables efficient collaboration and communication among team members. By customizing the site’s look and feel, businesses can promote their brand and create a professional online presence that meets their specific requirements.

From a developer’s perspective, site design is a group of site scripts.

How to add site design using pnp powershell

  1. Create a site script as mentioned in SharePoint Online Site Script

    Connect-PnPOnline -Url https://yourtenant.sharepoint.com/sites/yoursite -Interactive
$content=Get-Content "C:\Path\To\mysitescript.json" -Raw
Add-PnPSiteScript -Title "My Site Script" -Description "Custom site script" -Content $content -Version 1
Get-PnPSiteScript -Identity <SiteScriptID>

  2. Add site design using Add-PnPSiteDesign cmdlet

    Add-PnPSiteDesign -Title "My Company Design" -SiteScriptIds "e84dcb46-3ab9-4456-a136-66fc6ae3d3c5","6def687f-0e08-4f1e-999c-791f3af9a600" -Description "My description" -WebTemplate TeamSite -ThumbnailUrl https://contoso.sharepoint.com/sites/templates/siteassets/logo.png

To update/delete site design, you can use the following cmdlet

set-PnPSiteDesign -Identity <TenantSiteDesignPipeBind> [-Title <String>] [-SiteScriptIds <GuidPipeBind[]>]
 [-Description <String>] [-IsDefault] [-PreviewImageAltText <String>] [-PreviewImageUrl <String>]
 [-WebTemplate <SiteWebTemplate>] [-Version <Int32>] [-ThumbnailUrl <String>] [-DesignPackageId <Guid>] [-Connection <PnPConnection>]
 [<CommonParameters>]

 Remove-PnPSiteDesign [-Identity] <TenantSiteDesignPipeBind> [-Force] [-Connection <PnPConnection>]
