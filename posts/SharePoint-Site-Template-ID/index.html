SharePoint Site Template ID

Apr 1, 2023 13 min readSharePoint

A SharePoint site template ID is the unique identifier for each SharePoint site template. It is typically a string that can be accessed through the SharePoint object model or by using the site template gallery in the SharePoint admin center.

Using a SharePoint site template ID, you can:

  1. Create new SharePoint sites based on pre-defined templates. You can choose an existing template and use its ID to create a new site without having to create it from scratch.
  2. Create site collections from templates. Site collections are containers that contain multiple subsites. By creating a site collection from a template, you can quickly create an entire site hierarchy that includes multiple subsites.
  3. Automate the creation of sites or site collections in code. Using the SharePoint object model, you can write code to create new SharePoint sites or site collections without having to manually perform these processes. To create a site or site collection, you need to specify the template ID to use.
